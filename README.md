## Installation

Laravel 8

Copy file .env

```sh
cp .env.example .env
```

Install dependencies

```sh
composer install
```

Generate key

```sh
php artisan key:generate
```

Create database 
```sh
test_akupintar
```

Migrate and seed database
```sh
php artisan migrate --seed
```

Run local serve laravel
```sh
php artisan serve
```
